## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/jakeblackburnn/image-editor.git
   ```
2. Navigate to the project directory:
   ```bash
   cd image-editor
   ```

---

## Usage

run the program in add / view mode using cargo:

```bash
cargo run -- add
cargo run -- view
```
